YelpCamp

Description:

• Campsite booking app built as part of a full-stack web development course on Udemy.
• Hosted on MongoDB Atlas.
• Full CRUD capability & Authorisation.
- Create campsite listings with images, descriptions, and pricing.
- View created campsites and leave comments.
- Update your campsite details.
- Delete your created campsites.

Built on:

  • HTML
  • CSS (Bootstrap)
  • Node.js
  • Express
  • MongoDB
